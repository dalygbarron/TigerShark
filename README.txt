Tiger Shark Engine: Extreme Sex Edition
=======================================
This game engine will destroy your brain and give you dementia. It makes games
so good you will DIE.

How to Build
============
meson setup build
meson compile -C build

then you will have it in build/main. However, you will need to have installed
sfml 2.5, spdlog, pugixml, opengl, glm, openal, and catch2 to build and run the
tests.

If you do wanna run the tests then say meson test -C build and it will run them
wahooo.
